In this folder you can find all the deliverables related to the I&T project.
There are two zip files, named "DreamServer.zip" and "DREAMApplication.zip", that correspond, respectively, to the Maven project of the application 
server and to the Xcode project of the mobile application.
Furthermore, there is a pdf document, named "ITD1.pdf", which is the Implementation and Test document.
Finally, there is a folder, named "ApplicationDemo" which contains three demonstrative videos, that show the functioning of the mobile application.
